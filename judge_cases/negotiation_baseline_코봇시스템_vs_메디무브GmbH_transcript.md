# 협상 로그 [baseline] — 코봇시스템(판매) ↔ 메디무브 GmbH(구매)

## R1 판매자
기존 휠체어 설계를 바꾸지 않고도 계단과 턱을 넘게 하는 미라클휠이 귀사의 제품 경쟁력을 어떻게 높일 수 있을지 CES 2025 혁신상을 입증한 데모 영상으로 먼저 공유해 드리고자 합니다. 초기 도입 부담을 낮추기 위해 소량 PoC부터 시작 가능하며, 귀사의 검토를 위해 관련 기술 자료를 먼저 보내드려도 괜찮을까요?

## R1 구매자
제안하신 기능은 당사 신모델 차별화에 매우 흥미로운 요소입니다. 다만, 당사는 유럽 내 재활기기 제조사로서 CE MDR 규제 준수가 가장 중요한 선결 조건입니다. 귀사의 모듈이 유럽 의료기기 규정상 어떤 등급으로 분류되며, 기존 휠체어의 인증 유지에 영향을 미치지 않는지 관련 기술 문서를 먼저 확인하고 싶습니다.

## R2 판매자
유럽 시장 진출을 위한 CE MDR 준수와 기존 인증 유지의 중요성을 깊이 공감합니다. 현재 MDR 분석을 진행 중이며, 귀사의 인증 담당자가 검토할 수 있도록 기기 등급 분류 및 기존 휠체어 설계 변경 최소화에 관한 기술 자료를 정리해 전달드리겠습니다. 이 자료를 먼저 확인하신 후, 당사의 기술적 대응 범위를 직접 판단해 보시겠습니까?

## R2 구매자
MDR 대응 현황을 정직하게 공유해주셔서 감사합니다. 기술 자료를 보내주시면 저희 규제팀과 함께 해당 모듈이 기존 휠체어의 인증 범위 내에서 통합 가능한지, 아니면 추가 임상이나 별도의 인증 절차가 필요한지 면밀히 검토하겠습니다. 자료 검토 후, 3개월 내 부품 확정이라는 저희의 제품 개발 사이클에 맞추어 PoC를 진행할 수 있을지 최종 판단하겠습니다. 자료를 전달해주십시오.

## R3 판매자
규제팀 검토를 위해 MDR 분석 자료와 함께, 기존 휠체어 인증 범위 내 통합 가능성을 입증할 기술 백서를 동봉하겠습니다. 3개월 개발 사이클 내에 PoC를 완료하실 수 있도록, 귀사 규제팀이 즉시 검토 가능한 핵심 기술 문서들을 우선적으로 정리해 보내드려도 되겠습니까?

## R3 구매자
기술 백서와 MDR 분석 자료를 보내주시면 규제팀 검토를 즉시 시작하겠습니다. 특히 기존 휠체어의 프레임 강성이나 무게 중심 변화가 당사의 기존 CE 인증에 미치는 영향이 핵심입니다. 자료를 검토한 후, PoC의 구체적인 범위와 기술적 타당성을 논의하겠습니다.

## R4 판매자
프레임 강성과 무게 중심 변화가 기존 CE 인증에 미치는 영향은 제품 안전의 핵심임을 충분히 이해했습니다. 해당 데이터를 포함한 기술 백서와 MDR 분석 자료를 규제팀에서 즉시 검토하실 수 있도록 패키징하여 오늘 중으로 보내드리겠습니다. 이 자료를 통해 내부 검토를 진행하신 후, 추가로 필요한 기술적 근거가 있다면 말씀해 주시겠습니까?

## R4 구매자
데이터 패키징을 오늘 중으로 전달해 주시겠다는 점, 정직하고 신속한 대응으로 판단합니다. 규제팀 검토 시 가장 우려하는 지점은 모듈 장착 후의 동적 안정성 테스트 결과입니다. 해당 데이터가 포함된 기술 백서를 수령하는 대로 면밀히 분석하여 PoC 진입 여부를 결정하겠습니다.

## R5 판매자
동적 안정성 테스트 데이터가 규제팀 검토의 핵심임을 확인했습니다. 해당 수치와 분석 결과가 포함된 기술 백서를 오늘 중으로 송부하여 귀사의 PoC 결정 과정을 지원하겠습니다. 자료를 검토하신 후, 규제팀 차원에서 추가로 필요한 데이터가 있다면 편하게 말씀해 주시겠습니까?

## R5 구매자
오늘 중으로 송부해주시는 기술 백서와 동적 안정성 데이터를 규제팀과 즉시 검토하겠습니다. 해당 자료가 당사의 CE MDR 인증 범위를 유지할 수 있음을 증명한다면, PoC의 구체적인 범위와 일정을 바로 논의할 준비가 되어 있습니다. 자료 확인 후 추가적인 기술적 질의가 있을 수 있으니 그 점 양해 바랍니다. 자료를 기다리겠습니다.

## R6 판매자
규제팀 검토의 핵심인 동적 안정성 테스트 데이터와 기술 백서를 포함한 패키지를 방금 송부 완료했습니다. 귀사의 내부 검토 과정에서 기술적 근거가 추가로 필요하실 경우, 언제든 규제팀과 직접 소통할 수 있도록 지원하겠습니다. 검토 후 PoC 논의를 위한 다음 단계가 필요하시면 말씀해 주시겠습니까?

## R6 구매자
송부해주신 자료는 방금 수령하여 즉시 규제팀에 전달했습니다. 동적 안정성 테스트의 구체적인 수치와 MDR 대응 논리가 당사 내부 기준을 통과하는지가 PoC 승인의 결정적 변수입니다. 검토 결과에 따라 추가 질의가 있을 수 있으니, 그때 다시 논의하도록 하겠습니다. 자료 확인까지 잠시만 기다려 주십시오.

## R7 판매자
규제팀의 검토 과정을 충분히 존중하며, 결과를 기다리는 동안 추가로 필요하신 기술적 보완 사항이 있다면 언제든 말씀해 주십시오. 내부 의사결정을 돕기 위해 필요한 추가 자료가 있다면 즉시 준비하겠습니다. 검토가 완료되는 시점에 맞춰 PoC 논의를 이어가겠습니다.

## R7 구매자
보내주신 기술 백서와 테스트 데이터를 규제팀에 전달했습니다. 현재 팀에서 귀사의 데이터가 당사의 CE MDR 인증 범위를 유지하면서 구조적 무결성을 보장하는지 정밀 분석 중입니다. 내부 검토가 완료되는 대로 PoC 진행을 위한 기술적 타당성 여부를 회신하겠습니다. 결과가 나올 때까지 잠시만 기다려 주십시오.
