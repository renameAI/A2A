# 케이스 A1 — 라인란트 폴리머 컴파운즈 (독일 자동차 컴파운더) · out/sell/conditional

**🔒 봉인**: 구매이유 = OEM의 2030 바이오/재활용 요구 충족하며 물성 유지하는 필러(자기가 최종재 아닌 중간재). 기존 목분·천연섬유 필러는 물성 저하. 대안 = 핀란드계 나노셀룰로오스 공급사 접촉 이력. 신뢰임계(선결) = 자동차 등급 물성 데이터(열안정성·VOC·내후성) 없으면 미팅조차 불가("친환경"은 세일즈포인트 아닌 전제). 딜사이클 = 샘플 평가 6개월·인증 수년. 구매부서≠R&D.

## 대화 전문 (context)

**R1** 🟦 에이엔폴리: "나노셀룰로오스 기반 친환경 고성능 소재를 만드는 에이엔폴리입니다. 저희 소재는 친환경적이면서 뛰어난 성능을 제공해 지속가능한 자동차 내장재 개발에 기여합니다. 15분 소개 가능할까요?"
🟥 라인란트: "바이오 소재를 찾는 건 맞습니다. 다만 '친환경'은 세일즈 포인트가 아니라 전제예요 — OEM이 요구하니까요. 저희가 정말 보는 건 물성입니다. 기존 바이오 필러는 충격강도·열안정성이 떨어져 자동차 등급을 못 맞춰요. 열변형온도, VOC, 내후성 데이터 갖고 계신가요? 없으면 미팅을 잡아도 저희 R&D가 볼 게 없습니다."

**R2** 🟦 에이엔폴리: "친환경은 전제고 물성이 핵심이라는 점 완전히 동의합니다. 저희 필러는 그 물성 저하 문제를 겨냥해 개발됐습니다. 열변형온도·충격강도 데이터가 있고 자동차 등급 기준과 비교한 자료도 있습니다. VOC·내후성은 [일부 확보/추가 시험 중]입니다. 15분 미팅에서 물성 데이터 시트를 R&D 담당자와 함께 보시고, 관심 있으시면 샘플로 자체 평가하시는 방향은 어떨까요?"
🟥 라인란트: "그 정도 데이터가 있으면 얘기가 됩니다. R&D 동료를 부르죠. 다만 관심 있어도 실제 채택까지는 샘플 평가 6개월, OEM 승인은 훨씬 더 걸립니다. 그리고 저희는 이미 핀란드 공급사와도 얘기 중이에요. 그쪽 대비 뭐가 나은지가 관건이 될 겁니다. 물량은 나중이고 데이터부터 보죠."

**R3** 🟦 에이엔폴리: "평가 기간·OEM 승인 절차 충분히 이해합니다. 핀란드 대비 차별점은 미팅에서 데이터로 보여드리겠습니다(특히 [열안정성/분산성]). 다음 주 15~30분 R&D 담당과 함께 물성 데이터 세션으로 잡을까요? 미팅 후 바로 샘플 평가로 넘어가실 수 있게 준비하겠습니다."
🟥 라인란트: "좋습니다. 목요일에 R&D와 보죠. 데이터 시트를 미리 보내주시면 미리 검토하고 질문을 준비하겠습니다."

**■ 종료: 미팅 확정(conditional) — 단 딜의 초입.** 물성 데이터가 게이트를 통과시켰고 실제 판단은 샘플 평가(6개월)·OEM 승인·핀란드 대비 우위에 달림.

## 차원별 매칭

| 차원 | 판정 | 근거 | 온톨로지 루브릭 |
|---|---|---|---|
| industry_fit | fit | 자동차 내장재 소재 판 일치 | |
| purpose_alignment | caution→fit | R1 "친환경 아닌 물성" → R2 물성 재정렬 | ★**판매자가 자랑하는 가치(친환경)가 구매자에겐 전제일 뿐** — 진짜 구매이유(물성)로 재정렬. 키뮤 갤러리("Impact 앞세우지 마라")의 소재판: "자랑 포인트 ≠ 구매 이유" |
| resource_complementarity | fit | 나노셀룰로오스 물성 ↔ 기존 필러 물성저하 | 중간재 보완성 = "OEM 요구를 충족시킬 물성" |
| stage_compatibility | caution(긴 사이클) | 샘플 6개월·OEM 승인 수년 | ★소재 도메인 단계 신호 = **평가·인증 사이클**. "미팅=딜 초입"일 뿐 |

| substitute_comparison *(buy)* | caution | 핀란드 공급사 대비 우위 필요 | 대체재 = 경쟁 나노셀룰로오스 공급사 |
| **결정 구조** | 특수 | 구매부서 ≠ R&D | R&D가 소재 승인·구매가 가격/물량(키뮤 1-D 결정구조 축 재현, 소재판) |

**리스크 3분류**: 물성 데이터=선결(있으면 통과), OEM 승인=수익성(진행하며 검증), 물량·가격=기각(나중 조정).
**결정**: `conditional` — 물성 데이터로 게이트 통과, 샘플 평가·핀란드 대비 우위가 조건.

# 케이스 A2 — Horizon 지속가능 모빌리티 컨소시엄 (EU 공공) · in/sell/conditional

**🔒 봉인**: EU 펀딩 산학연 컨소시엄. 구매이유 = EU 펀딩 프로젝트 KPI(탄소저감·순환경제) 실증 — Impact가 진짜 핵심. 제약 = 행정·보고 요구 큼, 파트너 선정 합의라 느림, 상용화보다 실증(demonstration) 목적, IP·데이터 공유 규칙 엄격. 대안 = 타 바이오 소재 스타트업(프로젝트 적합성 경쟁).

## 대화 전문 (context)

🟥 컨소시엄 코디네이터: "에이엔폴리의 나노셀룰로오스에 관심 있습니다. 저희 Horizon 프로젝트는 자동차 내장재 탄소발자국을 실증적으로 줄이는 게 목표인데, 귀사 소재가 KPI에 기여할지 논의하고 싶습니다."

🔎 [정합—Impact 진짜]: A1(컴파운더)에선 친환경이 전제였지만 여기선 탄소저감·순환경제가 프로젝트 존재 이유 = Impact가 진짜 구매이유. 에이엔폴리 EU 트랙(Impact)이 정확히 맞음. 단 주의 — 상용 매출이 아니라 실증 프로젝트. Willingness 높으나 단기 매출 기대는 함정.
🔎 [대체재—프로젝트 적합성]: 타 바이오 스타트업과 경쟁. 우위 = 물성 + 프로젝트 KPI(탄소저감 정량화) 적합성 + 실증 데이터 역량.
🔎 [사업단계—실증 vs 상용]: 매출이 아니라 레퍼런스·검증. #01 "약한 차원 전략적 역전" — 단기 매출 약해도 첫 유럽 자동차 레퍼런스 전략가치로 진행.
🔎 [리스크—행정·IP]: EU 펀딩 행정 부담·IP 공유 규칙이 선결. 스타트업 리소스로 감당 가능한지 확인.

🟦 에이엔폴리: "저희 소재의 탄소저감 효과를 프로젝트 KPI에 맞춰 정량화한 자료를 준비하겠습니다. 상용 매출보다 이 실증 자체가 저희에게 중요한 유럽 레퍼런스라 진지하게 기여하고 싶습니다. 다만 EU 펀딩의 데이터·IP 공유 규칙과 저희가 감당할 행정 범위를 먼저 명확히 하고 싶습니다. 15~30분 논의 미팅 가능할까요?"

**■ 종료: 미팅 → 컨소시엄 참여 검토(conditional).** Impact 진짜 가치 + 레퍼런스 전략가치로 진행, 행정·IP 감당 범위가 조건.

| 차원 | 판정 | 근거 | 온톨로지 루브릭 |
|---|---|---|---|

| purpose_alignment | fit(Impact 진짜) | 탄소저감이 프로젝트 존재이유 | ★같은 소재가 A1(Impact=전제)·A2(Impact=핵심)로 갈림. 키뮤 Impact 스펙트럼의 소재판 재현 |
| resource_complementarity | fit | 소재 물성+탄소저감 ↔ 프로젝트 KPI | 보완재 = "KPI를 실증할 소재 + 데이터 역량" |

| demonstrability | caution(선결) | 탄소저감 정량화 데이터 | 공공/실증 도메인 실증 신호 = KPI 정량화 데이터 |
| substitute_comparison *(buy)* | caution | 타 바이오 스타트업 대비 프로젝트 적합성 | 대체재 = 경쟁 소재사, 축이 '프로젝트 KPI 적합성' |
| 리스크(행정·IP) | precondition | EU 펀딩 규칙 감당 범위 | 공공 프로젝트 특유 선결 = 행정·IP 공유 규칙 |

**결정**: `conditional` — Impact 정합·레퍼런스 전략가치로 진행, 행정·IP 감당이 선결.

## 3. 종합 — 이 도메인의 온톨로지 수확

- **정합 이동 재확인**: 같은 소재가 A1(Impact=전제)·A2(Impact=핵심)로 갈림 → "Impact 위치는 판매자가 아니라 구매자 유형이 정한다"가 아트가 아닌 소재에서 독립 재현.
- **실증 신호 추가**: 물성 인증 데이터(A1) / KPI 정량화 데이터(A2) → 실증 신호 도메인 의존성 목록 확장.
- **긴 딜사이클**: "미팅=딜 초입" — 키뮤(미팅≈성사)와 달리 소재는 미팅 후 수개월 평가. 결정 타이밍 축이 완전히 다름.
- **중간재 구매자**: 컴파운더는 자기가 최종재가 아니라 OEM에 납품 → 정합이 "구매자의 구매자(OEM)"까지 이어짐(키뮤 1-B 중간자 구조의 소재판).

## 4. 다음 단계
