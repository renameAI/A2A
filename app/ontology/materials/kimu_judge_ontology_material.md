# 키뮤스튜디오 협상 시뮬레이션 — Judge 온톨로지 개발 재료
### 대화 전문 + 차원별 매칭 + Context · 10개 케이스 (v1.0)

> **수신**: AI 박사님 (Judge 온톨로지·학습 데이터 설계)
> **목적**: 동일 판매자(키뮤스튜디오, 발달장애 디자이너 아트웍)를 구매자 유형·방향·렌즈·결말을 바꿔가며 협상 시뮬레이션한 결과를, 기획서 7장(Judge)의 **7차원 온톨로지에 매핑**한 재료. 각 케이스 = 대화 전문(context) + 차원별 판정·근거 + 온톨로지 루브릭.
> **연계**: 차원 정의 = 작성가이드 §2·기획서 7.2 / 결정 4종·리스크 3분류 = 기획서 7장 / 협상 = 7-A / 협상 내 explore-exploit = 7-A.7(검증 예정) / 가설 카드 = 7-B·스키마 v1.1 / 두 렌즈 = 7.11.

## ⚠️ 0. 이 재료의 유효 범위와 한계 — 먼저 읽어주세요

**이것은 학습 데이터(벽돌)가 아니라 온톨로지 설계도입니다.** 그대로 SFT에 넣으면 과적합합니다. 세 가지 이유:

1. **판매자가 하나뿐**: 10개 전부 키뮤가 판매자. 보편 차원이 아니라 "키뮤 특수 패턴"을 배울 위험(작성가이드 §0·§8의 "호텔 봇" 문제의 축소판 — 여기선 "키뮤 봇").
2. **구매측이 전부 `[구성]`**: 상대 기업·private state는 AI(Claude)가 생성한 개연적 시뮬레이션. 실제 BD 전문가의 관찰이 아님 → 자기 생성 데이터 학습 = 에코 챔버 위험(기획서 10.6).
3. **정답 앵커 없음**: 결말이 전부 시뮬레이션 산출. 실제 성사/거절 outcome(객관 체계, 평가검증_설계 §1)이 0.

**그러므로 이 재료의 올바른 용도**:
- ✅ 협업목적 정합·자원 보완성·실증 가능성 등 **각 차원에 어떤 판정 기준·신호가 들어가는지**(온톨로지 스키마·enum·루브릭) 설계.
- ✅ 리스크 3분류, 결정 4종, 거절 유형, 진위 게이트가 **실제로 어떻게 발현되는지**의 형판.
- ✅ "보편 차원 + 산업별 신호" 2층 구조의 신호 목록 수집(§ 종합지도).
- ❌ 이대로 학습 투입 금지. 학습용 벽돌은 §말미 "다음 단계"의 실데이터 경로로 조달.

**과적합 방지 장치(이 10개에 적용됨)**: 판매자는 고정이나 결말(성사/결렬/추격중단/hold/PoC)·방향(아웃/인바운드)·렌즈(sell/buy)·결정(4종 전부)·거절유형(풀림/deal-breaker/가치충돌/진위)을 **의도적으로 분산**. 단일 서사("미온→소구점교체→성사")를 깨는 것이 목적.

## 1. Judge 7차원 정의 (기획서 7.2 · 작성가이드 §2)

각 차원은 `fit / caution / unfit` 판정 + 근거 1문장 이상. 차원 간 불일치는 확인 리스크로 변환(JDG-03).

**공통 5차원 (sell·buy 모두)**
| enum 키 | 차원 | 묻는 것 |
|---|---|---|
| `industry_fit` | 산업 적합성 | 문제 도메인이 겹치는가 (업종 라벨 ≠ 도메인) |
| `purpose_alignment` | 협업목적 정합 | 상대가 이걸 원하는가 (want) |
| `resource_complementarity` | 자원 보완성 | 내 것이 상대 결핍을 메우는가 (fit) |
| `stage_compatibility` | 사업단계 호환 | 규모·예산·타이밍이 현실적인가 |
| `demonstrability` | 실증 가능성 | 증명할 신호·레퍼런스가 있는가 |

**buy 렌즈 전용 +2차원**
| enum 키 | 차원 | 묻는 것 |
|---|---|---|
| `substitute_comparison` | 대체재 비교 | 기존 대안보다 나은가 (상대평가) |
| `opportunity_cost` | 기회비용 | 수용 시 무엇을 포기하나 |

**인바운드 전용 앞단**
| `authenticity_gate` | 진위 게이트 | (인바운드) 상대가 실재·진짜인가 — 모든 차원에 선행 |

**결정 4종**: `recommend` / `conditional` / `hold` / `terminate`
**리스크 3분류**: 선결조건(precondition) / 수익성(profitability) / 기각(dismissed)

# 케이스 1 — 메종 클레르 (프랑스 리빙 브랜드) · out/sell/recommend

**🔒 봉인**: 구매이유 = 디자인 신선도 55% > 가격 30% > CSR 15%(CSR 앞세우면 미온). 대안 = 유럽 아티스트 풀 풍부(협상력 카드). MG 거부. 실물 샘플 필수. 콜라보는 파리 에이전시 경유 관행.

## 대화 전문 (context)

**R1** 🟦 키뮤: "발달장애 디자이너 아트웍 스튜디오 키뮤입니다. 저희 아트웍으로 홈·리빙 캡슐을 만들면 포용과 다양성이라는 브랜드 가치를 제품으로 구현하실 수 있습니다. 15분 미팅 가능할까요?"
🟥 메종 클레르 *(미온)*: "CSR 가치엔 공감해요. 그런데 저희 콜라보 라인은 매대에서 팔려야 하는 상업 라인이라 바이어가 먼저 보는 건 '이 패턴이 고객 거실에 놓일 만큼 매력적인가'입니다. 포트폴리오는 흥미롭게 봤지만 지금 미팅 우선순위인지는 모르겠네요. 그리고 콜라보는 보통 파리 에이전시를 통해 진행합니다."

**R2** 🟦 키뮤: "순서를 잘못 짚었네요. 저희 아트웍은 그 자체로 팔리는 디자인이 먼저이고 포용 서사는 보너스입니다. [국내 브랜드] 콜라보에서 캡슐이 정규 라인보다 잘 나간 실적도 있습니다. 계약·자료 요청이 아니라 딱 15분만 셀렉션 보며 '메종 클레르 고객에게 팔릴 패턴인지' 감만 잡는 자리를 제안합니다. 에이전시 방식이 편하시면 미팅에서 조율하죠."
🟥 메종 클레르: "그 프레임이면 낫네요. 15분은 낼 수 있어요. 다만 ①유럽 아티스트 풀이 넉넉해서 아시아 직거래로 갈 이유가 분명해야 내부 설득이 되고 ②최종엔 실물 샘플을 봐야 계약합니다. 이 두 개를 미팅에서 납득시켜 주시면 잡죠."

**R3** 🟦 키뮤: "직거래 이유는 — 저희 아트웍은 유럽 풀에 없는 결이고 에이전시를 끼면 독창성이 표준화됩니다. 직거래면 전용 패턴을 만들 수 있어요. 실물은 대표 패턴 3종을 미팅에 지참하겠습니다. 다음 주 15~30분 잡을까요?"
🟥 메종 클레르: "좋습니다. 실물을 가져온다면 얘기가 되죠. 화요일 오후로 잡죠."

**■ 종료: 미팅 확정(recommend).** 미온을 소구점 교체로 전환, 대체재·실증은 미팅 어젠다로 이월.

## 차원별 매칭

| 차원 | 판정 | 근거 | 온톨로지 루브릭 |
|---|---|---|---|
| industry_fit | fit | 라이프스타일 리테일 × 아트 콜라보 판 일치 | 업종 라벨(스튜디오 vs 브랜드)이 달라도 문제 도메인 겹침 |
| purpose_alignment | caution→fit | R1 미온의 뿌리, R2 소구점 교체로 해소 | ★판매자 정체성(임팩트)이 아니라 구매자 구매이유(팔릴 디자인)로 판정. 미온이 '방향' 정정이면 정합, '증거' 요구면 실증 |
| resource_complementarity | fit | 독창 아트 ↔ 브랜드 콜라보 신선도 갈증 | 정합과 분리 — 원함✓+메움✓ 둘 다 성립 |
| stage_compatibility | caution(이월) | SS 4개월 타이밍 | 콜라보 도메인 단계 신호 = 시즌 캘린더. 낮은 CTA로 미팅 이월 가능 |
| demonstrability | caution→해소경로 | 실물 요구 → 3종 지참 | 디자인 도메인 실증 신호 = 실물 프린트 퀄리티(이미지 ≠ 증거) |
| substitute_comparison *(buy)* | caution | 유럽 풀 풍부(협상력 카드) | buy가 활성화 — 가격 아닌 '직거래 정당화' 요구 형태 |
| opportunity_cost *(buy)* | fit(완화) | 시즌 슬롯 희소하나 MG 제거로 다운사이드↓ | MG 제거 = 기회비용 완화 손잡이(#02 원복보장 계열) |

**7-A.7 관찰**: R1 explore("미온=정합" 베팅) → R2 소구점 교체 → 상대 긍정 = 확증 → 정합 exploit 승격.

# 케이스 1-B — 아틀리에 뇌프 (광고대행사) · out/sell/recommend

**🔒 봉인**: 구매이유 = "지금 피칭 중 클라이언트 건에 꽂히는가" 65%. 실제 뷰티 포용 캠페인 준비 중. 대안 = 자체 아트디렉션 팀. IP·사용권 clear 결정적. 토크니즘 극도 경계. 맞으면 빠름.

## 대화 전문 (context)

**R1** 🟦 키뮤: "에이전시 브랜드 캠페인에 저희 아트웍을 활용하시면 독창 비주얼 + 진정성 있는 다양성 스토리를 동시에 얻으실 수 있습니다. 15분 소개 가능할까요?"
🟥 아틀리에 뇌프: "포트폴리오는 인상적이네요. 근데 저희는 자체 아트디렉션 팀과 상시 일러스트레이터가 있어서 외부를 굳이 쓰려면 '왜 우리 자원 대신'이 분명해야 합니다. 범용 콜라보면 우선순위 아니에요. 특정 캠페인 유형을 염두에 두셨나요?"

**R2** 🟦 키뮤: "범용이 아니라 진행 중 캠페인에 맞춰 들어가는 게 맞습니다. 자체 팀 대신 쓰는 이유는 하나 — 저희 아트웍은 발달장애 디자이너가 실제로 그린 진짜 작품이라 '다양성'을 연출하는 게 아니라 실재합니다. 일러스트레이터가 흉내 낸 것과 실제 당사자 작품은 클라이언트 설득력이 다릅니다. 그런 결의 피칭을 준비 중이시라면 딱 15분 맞을지 보고 싶습니다."
🟥 아틀리에 뇌프: "…마침 뷰티 클라이언트의 포용 테마 캠페인을 준비 중입니다. '실제 당사자 작품'은 차별화 포인트가 되겠네요. 다만 ①클라이언트가 토크니즘 극도 경계('착한 척'은 역효과) ②상업이라 저작권 clear 필수. 이 둘 정리되면 미팅 잡죠."

**R3** 🟦 키뮤: "진정성은 — 디자이너가 창작 주체로 크레딧되고 로열티를 받는 구조라 '동원'이 아닌 '협업', 이게 토크니즘 방어막입니다. 저작권은 클리어런스 문서를 미팅에 지참하겠습니다. 다음 주 15~30분, 그 캠페인에 맞을지 보는 자리로 잡을까요?"
🟥 아틀리에 뇌프: "좋습니다. 목요일로 잡죠. 포용 테마 셀렉션을 미리 추려 오시면 시간을 아끼겠네요."

**■ 종료: 미팅 확정(recommend).** 진행 캠페인이라는 실니즈에 정확히 꽂혀 빠르게 수렴.

## 차원별 매칭

| 차원 | 판정 | 근거 | 온톨로지 루브릭 |
|---|---|---|---|
| industry_fit | fit | 아트웍 활용 판 맞음 | |
| purpose_alignment | caution→fit | R1 "특정 캠페인?" → R2 떠보기로 확증 | ★정합 기준이 최종소비자가 아니라 **에이전시의 현재 피칭**에 있음. 중간자 구매자는 "구매자의 구매자"로 정합 이동 |
| resource_complementarity | fit | R2 "진짜 vs 연출"로 명시화 | 대행사 보완성 = 자체 일러스트로 대체불가한 '실재성' |
| stage_compatibility | fit(촉매) | 피칭 데드라인이 순풍 | ★같은 차원이 정반대 부호 — 시즌캘린더(제약)와 달리 피칭데드라인은 촉매 |
| demonstrability | caution→해소 | IP·사용권 clear 요구 | 대행사 실증 신호 = **저작권·상업 사용권 clear**(품질 아님) |
| substitute_comparison *(buy)* | caution→우위 | 자체 아트디렉션 팀 대비 실재성 | 사내 자원이 대체재 |
| opportunity_cost *(buy)* | fit(낮음) | 캠페인별 예산, 슬롯 압박 약함 | |

**7-A.7 관찰**: R2 "진행 캠페인 있나" 떠보기(explore) → 뷰티 캠페인 노출 = 확증 → 정합 승격.

# 케이스 1-C — 갤러리 뤼미에르 (컨템포러리 갤러리) · out/sell/conditional

**🔒 봉인**: 구매이유 = 예술적 진정성 50% > 관객반응 30% > 사회서사 20%. 장애를 마케팅으로 소비하는 것에 극도 거부(아트월드 토크니즘 경멸). 대안 = 월 수십 포트폴리오. 작가 개인이 주체여야. "우리는 아트샵 아니다"(전시 먼저). 전시 6~12개월 전 확정.

## 대화 전문 (context)

**R1** 🟦 키뮤: "저희 작가들 작품으로 갤러리와 콜라보 상품이나 전시를 함께하면 포용과 다양성이라는 의미 있는 메시지를 관객에게 전할 수 있습니다. 15분 미팅 가능하실까요?"
🟥 갤러리 *(경계)*: "'포용·다양성 메시지'나 '콜라보 상품'이라는 표현은 오히려 신중하게 만듭니다. 저희는 아트샵이 아니라 작가를 대표하는 갤러리예요. 저희가 보는 건 작품이 동시대 미술 맥락에서 어떤 고유성을 갖는가입니다. '발달장애'를 앞세우면 작품이 아니라 사연으로 소비된다는 인상을 줍니다. 개별 작가의 작품 세계 이야기라면 다르지만요."

**R2** 🟦 키뮤: "잘못된 언어로 접근했습니다. 저희가 하고 싶은 건 메시지 캠페인이 아니라 개별 작가들의 작품 세계를 동시대 미술 맥락에서 보여주는 것입니다. 한 작가는 [반복 패턴·색채 언어]로 자기 시각 문법을 구축했는데 아웃사이더 아트·미니멀리즘 계보에서 논의될 여지가 있습니다. '발달장애'는 배경일 뿐 앞세울 라벨이 아닙니다. 딱 15분 작가 두세 명 작품을 순수하게 작품으로만 보시고 갤러리 방향과 맞을지 봐주실까요? 상업 얘기는 일절 빼고요."
🟥 갤러리: "다르게 들리네요. 작가 개인의 시각 언어로 접근하신다면 저는 늘 새 작가를 봅니다. 다만 월 수십 개 포트폴리오 중 전시는 극소수예요. '왜 이 작가가 지금 우리 갤러리에서 보여져야 하는가'라는 큐레토리얼 질문에 답이 돼야 합니다. 진행한다면 전시가 먼저, 굿즈는 이후 부수적으로만. 이 전제가 괜찮으면 이미지 보내주시면 15분 통화로 시작하죠."

**R3** 🟦 키뮤: "전시가 먼저라는 전제 동의합니다. '왜 지금 이 작가'도 미팅에서 다루겠습니다. 미리 답하면 — 미술계에서 뉴로다이버시티·아웃사이더 아트 큐레토리얼 관심이 커지고 있고, 저희 작가들은 훈련된 미술 언어에 오염되지 않은 고유 형식을 보여줍니다. 갤러리 신진 발굴 방향과 만나는 지점입니다. 이미지 오늘 보내고 다음 주 통화로 큐레토리얼 핏을 보시죠. 안 맞으면 솔직히 아니라 하셔도 좋습니다."
🟥 갤러리: "그 정도 태도면 볼 가치가 있어요. 이미지 보내주시면 통화하죠. 단 봐서 안 맞으면 정중히 거절하겠습니다."

**■ 종료: 통화 확정, 조건부·저확신(conditional).** Impact 서사가 역신호로 작동, 소구 전면 전환으로 겨우 문을 엶.

# 케이스 1-D — 노르딕일렉트로닉스 CSR팀 (가전 ESG) · out/sell/recommend

**🔒 봉인**: 구매이유 = 임팩트 진정성 45% > 커뮤니케이션 활용도 30% > 실행 용이성 25%. 예산 = CSR(마케팅과 별도, 연간 확정, 지금 내년 기획). KPI = 수혜자 수·미디어 노출·임직원 참여. 대안 = 기존 NGO 후원. 워싱 극도 경계. 담당자 결정권 약함(마케팅·경영진 승인). 승인 다단계.

## 대화 전문 (context)

**R1** 🟦 키뮤: "저희 작품을 활용한 사회공헌 협업 프로그램을 제안드립니다 — 발달장애 예술가 자립을 지원하면서 그 스토리를 브랜드 ESG 활동으로 전하실 수 있습니다. 15분 소개 가능하실까요?"

**R2** 🟦 키뮤: "기부는 일회성이지만 저희 협업은 디자이너가 창작자로 참여하고 정당한 로열티를 받는 구조 — 시혜가 아니라 경제적 자립입니다. '얼마 기부'가 아니라 '몇 명이 자기 작품으로 소득을 얻었나'라는 측정 가능한 성과로 보고하실 수 있어요. 워싱 우려는 그 구조가 방어막입니다 — 로열티 배분을 투명 공개해 '실질 지원'을 숫자로 증명합니다. 내부에서 활용하실 자료 형태로 정리해 미팅 잡을까요?"
🟥 노르딕 CSR: "정확히 그게 제가 필요한 부분입니다. 제가 좋게 봐도 마케팅·경영진을 설득해야 하거든요. '수혜 예술가 수'·'로열티 지급액'·'임직원 참여 확장 가능성'이 자료에 있으면 도움됩니다. 그리고 경영진은 미디어에 어떻게 보일지를 봐요 — 브랜드 스토리로도 쓰일 수 있다면 통과 가능성이 높아집니다."

**R3** 🟦 키뮤: "자료를 이렇게 준비하겠습니다 — ①측정 가능한 사회적 성과(수혜자 수·로열티 구조) ②임직원 아트 워크숍 등 참여 확장안 ③브랜드 스토리 확장 시 미디어 앵글. 상업 콜라보는 빼고 사회공헌 프레임으로만 정리하겠습니다(예산 소관 문제 없도록). 다음 주 15~30분 잡을까요?"
🟥 노르딕 CSR: "내부에 가져가기 딱 좋겠어요. 예산 소관까지 신경 써주셔서 감사합니다 — 그게 실무에선 제일 골치거든요. 수요일 오후로 잡죠."

**■ 종료: 미팅 확정 + 담당자 우군화(recommend).** 단 "미팅 성사 ≠ 딜 성사" — 결정은 상급자에.

# 케이스 1-E — 웨스트필드 리테일 그룹 (미국 대형 리테일러) · out/sell/terminate

**겨냥**: 결렬① — 구조적 불가(선결조건 deal-breaker). **모든 미스매치가 소구점으로 풀리지 않는다.**

**🔒 봉인**: 구매이유 = 매대 회전율·마진. 선결조건 = ①Prop65+CPSIA 인증 ②500매장 양산 캐파·물류 ③EDI 연동 ④배상책임보험. 협상 여지 거의 없음(표준 벤더 계약).

## 대화 전문 (context)

**R1** 🟦 키뮤: "저희 아트웍으로 홈·리빙 굿즈 라인을 만들어 매장에 입점하면 독창 디자인과 브랜드 스토리를 함께 제공할 수 있습니다. 15분 소개 가능할까요?"
🟥 웨스트필드: "미팅 전 서류로 먼저 확인하는 게 프로세스입니다. ①전 SKU에 Prop 65·CPSIA 인증 있으신가요? ②500개 매장 초도·리오더 감당할 양산 캐파 되나요? ③배상책임보험은요? 이 셋이 안 되면 디자인이 좋아도 시스템에 못 올립니다."

**R2** 🟦 키뮤: "솔직히 말씀드리면 인증·보험은 시간을 들이면 갖출 수 있지만 500개 매장 대량 공급 양산 캐파는 현재 규모로 무리입니다. 저희는 소규모·고부가 아트 협업이 강점이라 대형 체인 전매장 입점은 지금 단계에 안 맞습니다."
🟥 웨스트필드: "솔직히 말씀 감사합니다. 저희도 그 부분이 걱정이었어요. 캐파가 안 되면 진행이 어렵습니다. 나중에 규모를 갖추시거나 소량 한정 라인을 보신다면 그때 연락 주세요."

**■ 종료: 결렬(terminate/breakdown).** deal-breaker(양산 캐파) 구조적 미충족. 억지 성사 대신 정직한 철수 + 관계 보존.

## 차원별 매칭

| 차원 | 판정 | 근거 | 온톨로지 루브릭 |
|---|---|---|---|
| industry_fit | fit | 홈·리빙 판 맞음 | ★산업 fit이어도 딜은 죽는다 — 필요조건이지 충분조건 아님 |
| purpose_alignment | unfit | 회전율·마진 ↔ 고부가 소량, 방향 어긋남 | 대형 리테일러 구매이유 = 매대 경제성. 아트 스토리 무의미 |
| resource_complementarity | **unfit** | 소량·고부가 ↔ 대량·저마진 안 맞물림 | ★보완성 unfit이면 다른 차원 fit이어도 진행 불가 |
| stage_compatibility | **unfit(deal-breaker)** | 500매장 양산 캐파 구조적 미달 | ★**선결조건형 리스크** — 손잡이로 못 푸는 결렬 사유. 하드 차단 |
| demonstrability | 무관 | 캐파에서 이미 결렬 | deal-breaker 확정 시 나머지 차원 판정 무의미 |

**리스크 3분류**: 인증=선결(확보가능·시간), 캐파=선결(구조적 미달=deal-breaker), 보험=선결(비용).
**결정**: `terminate` — 사업단계 deal-breaker 하드 차단. **철수하되 관계 보존(규모 되면 재접촉).**

# 케이스 1-F — 노바트렌드 (패스트패션) · out/sell/terminate

**겨냥**: 결렬② — 가치충돌·안 풀리는 거절(카피). 1-E와 다른 종류: "못 함"이 아니라 **"하면 안 됨"**.

**🔒 봉인**: 속내 = 트렌드를 싸게 빨리. 위험 = 마음에 들면 자체 팀이 '유사하게' 그려 라이선스 없이 쓰는 카피 관행. 진정성 관심 거의 없음. 로열티에 극도 인색.

## 대화 전문 (context)

**R1** 🟦 키뮤: "저희 아트웍으로 캡슐 컬렉션을 만들면 독창 프린트와 다양성 스토리를 함께 얻으실 수 있습니다. 15분 미팅 가능할까요?"
🟥 노바트렌드: "아트웍 좋네요. 다양성 캡슐 시즌마다 하는데 관심 있어요. 근데 저희는 로열티는 잘 안 하고요, 보통 디자인을 일회성으로 사거나 저희 팀이 참고해서 자체 재해석하는 식이에요. 시안 몇 개 먼저 보내주시면 검토할게요."

**R2** 🟦 키뮤: "저희 모델을 분명히 말씀드리면, 저희 아트웍은 발달장애 디자이너가 창작 주체로 크레딧되고 지속 로열티를 받는 것이 핵심입니다. 일회성 매입이나 '재해석'은 저희가 지키려는 가치와 맞지 않습니다. 정식 라이선스 협업이 아니라면 시안 공유는 어렵습니다. 로열티 기반 정식 협업을 고려하실 여지가 있으실까요?"
🟥 노바트렌드: "저희 구조상 로열티는 어렵고요. 시안 몇 개 보고 핏 맞으면 얘기하려던 거였는데. 부담되시면 어쩔 수 없죠. 나중에 생각 바뀌면 연락 주세요."

**■ 종료: 추격 중단(terminate/walk-away).** 안 풀리는 거절(로열티 거부 + 카피 암시). 탐색예산 회수, 시안 미공유로 IP 보호.

## 차원별 매칭

| 차원 | 판정 | 근거 | 온톨로지 루브릭 |
|---|---|---|---|
| industry_fit | fit | 패션 프린트 판 맞음 | 산업 fit ≠ 진행 가능 |
| purpose_alignment | **unfit(가치충돌)** | 싸고 빠른 트렌드 ↔ 지속가능 창작자 소득 | ★정합 unfit 2종: ①방향 다름(1-E) ②**가치 충돌**(1-F) |
| resource_complementarity | 표면 fit | 아트웍은 매력적이나… | ★표면 보완성 있어도 **거래 구조가 착취적이면 진행 불가** |
| demonstrability | 무관 | 가치충돌에서 종료 | |
| **거절 유형** | 안 풀림 | 로열티 거부 + 카피 암시 | ★#03 '카피/의지' 유형. deal-breaker(요건 미충족)와 다름 — **상대 의도가 착취적**. 표면 Willingness(관심)가 위험 은폐 |

**결정**: `terminate` — 가치충돌 + 안 풀리는 거절. **시안 미공유(IP 보호)가 결정의 일부.**

# 케이스 1-G — 블룸카페 그룹 (프랜차이즈) · in/buy/conditional

**겨냥**: 인바운드 · 진위 게이트 통과 후 실질 판단 · Willingness 높음의 함정(선의지만 부적합).

**🔒 봉인**: 대표가 감동해서 먼저 연락(선의, 마케팅 계산 아님). 실재·검증 가능. 단 ①예산 지점당 소액 ②'전 지점'이나 가맹점이라 강제 못 함 ③아트 = 벽 장식(로열티 구조 애매) ④구체 실행계획 없음.

## 대화 전문 (context)

🟥 블룸카페 대표: "키뮤 기사 보고 감동했습니다. 저희 카페 20개 지점 전체에 작가님들 작품을 걸고 싶어요. 좋은 일에 함께하고 싶습니다. 어떻게 진행하면 될까요?"

🔎 [진위 게이트 — **통과**]: 실재 프랜차이즈, 대표 본인 공식 연락. → 이후 실질 차원 판정으로. (교훈: 진위 통과가 끝이 아니라 시작.)
🔎 [Willingness 함정]: 매우 높음(감동·즉시·전지점). 그러나 #04 — 상대가 더 원해도 실질 우선. 뜯어보면 정합(일회성 장식 ↔ 지속 소득)·단계(가맹점 실행력 불확실·소액)·실행(설계를 키뮤가 떠안음) 모두 약함.

🟦 키뮤: "먼저 연락 주셔서 영광입니다. 몇 가지만 여쭐게요 — ①20개 지점이 직영인가요 가맹점인가요? ②아트를 벽 장식으로 생각하시는지, 굿즈·시즌 캠페인 확장도 열려 있으신지? ③함께 설계할 담당자가 계실까요?"
🟥 대표: "대부분 가맹점이에요. 권유는 하지만 강제는 어렵죠. 굿즈·캠페인은 생각 안 해봤고 그냥 벽에 예쁘게 거는 걸 생각했어요. 담당자는 없고 제가 직접 챙기려고요."

🔎 [종합]: 진위 통과했으나 정합·단계·실행 caution. Willingness 높으나 실질 약함. 대표 선의는 진짜, 관계 가치 있음. → hold: 규모 재설계 역제안.

🟦 키뮤: "20개 전 지점을 한 번에 하기보다 직영 2~3곳에서 작은 시범 프로젝트로 시작하는 걸 제안드립니다. 벽 아트 + 시즌 한정 굿즈를 얹어서 반응 좋으면 가맹점으로 넓히는 방식이요. 작가님들께 지속 구조가 되고 대표님도 부담 없이 시작하실 수 있어요. 15분만 더 이야기 나눠볼까요?"
🟥 대표: "훨씬 현실적이네요. 20개 한 번에는 막막했거든요. 직영 3곳으로 시작하는 거 좋아요."

**■ 종료: 축소 PoC 합의(hold→conditional).** 진위 통과, Willingness 함정을 규모 축소로 관리, 정합을 굿즈 확장으로 보강.

## 차원별 매칭

| 차원 | 판정 | 근거 | 온톨로지 루브릭 |
|---|---|---|---|
| **authenticity_gate** | **통과** | 실재·공식 연락 | ★인바운드 앞단. **통과가 끝이 아니라 시작** — 이후 실질 차원을 봐야 |
| Willingness | very_high | 감동·즉시·전지점 | ★높은 Willingness ≠ 좋은 매칭. 상대가 더 원해도 실질 우선(#04) |
| purpose_alignment | caution | 선의(Impact)지만 일회성 장식 ↔ 지속 소득 | 인바운드에서도 '상대가 원함'과 '우리에게 필요한 구조'는 다를 수 있음 |

**결정**: `conditional` — 진위 통과 + Willingness 높으나 실질 약함 → 규모 축소 PoC 재설계. #04 "Willingness 높아도 실행 리스크로 규모 축소"의 인바운드판.

# 케이스 1-H — 메종 클레르 buy 렌즈 (케이스 1의 짝) · out/buy/recommend

# 케이스 1-I — 미도리야 백화점 (일본) · out/sell/hold

**겨냥**: hold 결정 · 미온 ≠ 거절 · 문화 미스매치. Judge 4결정 중 hold 실증.

**🔒 봉인**: 아트 콜라보 관심 있으나 의사결정 극도 신중·다단계(품의 문화). 담당자는 긍정이나 조직이 느림. 일본 리테일 = 품질·완성도·신뢰관계 중시, 검증 길고 재촉을 무례로 받아들임. 관심 있어도 절제적 표현(미온처럼 보이나 실제 거절 아님).

## 대화 전문 (context)

**R1** 🟦 키뮤: "저희 작품으로 아트 코너 콜라보나 시즌 팝업을 함께하면 좋겠습니다. 15분 소개 가능하실까요?"
🟥 미도리야: "소개 감사합니다. 작품 잘 봤습니다. 검토해보겠습니다. 다만 저희가 이런 건은 내부적으로 시간을 두고 보는 편이라 바로 답을 드리긴 어렵습니다. 자료를 좀 더 주시면 참고하겠습니다."

🔎 [미온 ≠ 거절]: 표면은 미온. 케이스 1(브랜드)에선 미온이 매력 문제였으나 — 여기선 "작품 잘 봤다" + 자료 요청 = **관심 있으나 문화적으로 절제된 표현 + 조직적으로 느림.** 매력 부족(소구점)으로 오판하면 안 됨. 성급히 소구점 바꾸거나 밀면 무례로 비침. → hold: 확인 리스크 남기고 관계를 천천히.

**R2** 🟦 키뮤: "충분히 시간을 두고 보셔도 좋습니다. 참고 자료를 정리해 보내드릴게요 — 품질 관리 방식, 납기 안정성, 다른 매장에서 아트를 품위 있게 선보인 사례. 급하게 결정하실 필요 전혀 없고, 궁금하시면 언제든 연락 주세요. 실물 샘플도 보내드리겠습니다."
🟥 미도리야: "그렇게 해주시면 내부 검토하기 좋겠습니다. 샘플은 보내주시면 품질을 직접 보고 판단하겠습니다. 시간이 걸리더라도 연락드리겠습니다."

🔎 [신호]: 샘플 수용 = 관심의 실제 신호(미온이 거절 아니었음 확인). 조직 속도는 여전히 느림. → hold 유지, 확인 리스크를 자료·샘플로 채우며 긴 신뢰 구축. 추격도 포기도 아님.

**■ 종료: hold(판단 보류·관계 지속).** 명확한 성사도 결렬도 아님.

## 차원별 매칭

| 차원 | 판정 | 근거 | 온톨로지 루브릭 |
|---|---|---|---|
| industry_fit | fit | 백화점 아트 코너 판 맞음 | |
| purpose_alignment | caution | 관심 있으나 확정 안 됨 | ★**미온 ≠ 거절 ≠ 정합 문제.** 문화적 절제 표현일 수 있음 — 표현 스타일과 실제 의사 구분 |
| stage_compatibility | caution | 조직 의사결정 느림·다단계 | 느린 속도를 '관심 없음'으로 오판 금지. 속도는 문화·조직 특성 |
| demonstrability | caution→진행중 | 품질·납기 검증 요구 | 일본 리테일 실증 = 품질 안정성·신뢰 관계(장기) |
| 신뢰 | caution | 새 해외 파트너 검증 길다 | 문화권별 신뢰 구축 속도 상이 — 재촉이 역효과 |

**결정**: `hold` — 관심 있으나 조직 속도·문화 검증으로 확정 불가. 성급한 recommend도 terminate도 아닌 판단 보류. **hold는 실패가 아니라 유효한 판단.**

# 케이스 1-J — Centre Lumière (프랑스 발달장애 사회복지센터) · in/buy/conditional

**겨냥**: buy 렌즈 · 비영리/공공 구매자 · 서비스형(커리큘럼) 보완재 · PoC 요청 · Impact 스펙트럼 최상단.

**🔒 봉인**: 센터가 키뮤의 발달장애 디자이너 육성 모델을 보고 아트/디자인 커리큘럼 PoC 요청(인바운드+buy). 구매이유 = ①이용자 자기표현·직업재활 다양화 ②기부자·공적지원 성과 ③장기 소득화. 제약 = 예산 제한(공적재원)·프랑스어 현지화 필요·발달정도별 개별화·성과측정 필요·다단계 승인. Willingness 높음(진정성).

## 추론 궤적 (context — 구매자 렌즈, 센터)

🔎 [자원 보완성]: 우리 문제 = 이용자 직업재활·자기표현 프로그램 다양화. 키뮤는 발달장애인을 실제 디자이너로 육성한 모델·커리큘럼 보유 → 우리 결핍(전문 아트/디자인 교육 역량)을 정확히 메움. 적합. (단 한국 기반 → 프랑스 이용자·문화 현지화 필요.)
🔎 [협업목적 정합]: 우리는 재활·자기표현·장기 소득화를 원함. 키뮤 미션(창작자 육성·자립)과 깊이 일치. ★여기서는 Impact/사회적 가치가 진짜 구매이유의 핵심 — 갤러리(1-C)와 정반대, CSR(1-D)보다 순수. 적합.
🔎 [대체재 비교]: 대안 = 일반 미술치료사·현지 아트강사 / 기존 재활 유지. 이들은 발달장애인을 전문 디자이너로 육성한 실증 모델 없음 — 키뮤 차별점은 재활을 넘어 직업화·소득화 경로. 단 현지 강사는 언어·문화 접근성 높음. → 차별화 분명하나 현지화 리스크와 트레이드오프.
🔎 [기회비용]: 제한 예산을 PoC에 쓰면 다른 프로그램 못 함. 그러나 PoC 소규모라 부담 작고, 성공 시 기부자·공적지원 성과로 이어져 재원을 늘릴 수 있음. → 낮음, 업사이드 있음.
🔎 [실증 가능성]: 공적 보고 위해 성과 측정 필요. "이용자 N명이 작품 완성, 자기효능감·직업기술 향상"을 측정 가능하게 설계해줄 수 있는가가 관건. ★이 도메인 실증 신호 = 재활·교육 성과 지표(판매실적도 IP도 아님).
🔎 [실행 — 선결조건]: ①프랑스어 커리큘럼·현지 강사 훈련(없으면 진행 불가=선결) ②발달정도별 개별화 ③예산 범위 설계. → PoC를 소규모(8~10명·8주)로 잡아 통제.

## 3. 종합 — 온톨로지 루브릭 수확 (박사님 설계 입력)

### 3.1 협업목적 정합 — "구매자의 구매자가 누구냐"에 따른 기준 이동
```
최종소비자(브랜드) → 중간 클라이언트(대행사) → 미적 판단자 개인(갤러리) → 
조직 ESG 목표(CSR) → 사회적 미션(비영리센터)
```
정합은 고정 기준이 아니라 "구매자가 이걸로 무엇을 하려는가"로 재정의. 중간자·조직·비영리 구매자에서 이동 큼.

### 3.2 Impact 스펙트럼 — 같은 자산의 Impact 위치 (판정 로직)
```
역신호 ←────────────────────────────────────────→ 1순위·핵심
갤러리      브랜드       대행사        CSR팀       사회복지센터
(앞세우면   (보너스)    (차별화무기,   (진짜 구매   (미션 자체)
 거부)                  '진짜'일때)    이유)
```
`impact_auto_first_checked`는 "체크했나"가 아니라 **구매자를 이 스펙트럼에 위치시키는 판정**.

### 3.3 실증 가능성 — 도메인별 신호 (보편차원 + 산업신호 2층 구조)
| 도메인 | 실증 신호 |
|---|---|
| 리빙 브랜드 | 실물 프린트 퀄리티 |
| 광고대행사 | IP·상업 사용권 clear |
| 갤러리 | 동시대 미술 담론 내 위치 |
| CSR/ESG | 수혜자 수·로열티 투명성 |
| 비영리/재활 | 재활·교육 성과 지표 |
| *(호텔 #01/#02)* | *리뷰·점유율·업셀링 데이터* |
→ 모델이 배울 것 = "실물 확인하라"(암기)가 아니라 "실증은 그 도메인 신뢰 신호로 검증하라"(구조).

### 3.4 결정 4종의 발현 조건
| 결정 | 조건 | 케이스 |
|---|---|---|
| recommend | 차원 대부분 fit, 확인리스크 관리가능 | 1·1-B·1-D·1-H |
| conditional | 강한 fit + 선결조건/규모조정 필요 | 1-C·1-G·1-J |
| hold | 관심 있으나 확정 불가(속도·문화·미확인) | 1-I |
| terminate | deal-breaker(구조적) 또는 가치충돌/안풀리는거절 | 1-E·1-F |

### 3.5 terminate의 두 유형 (구분 필수)
- **① 구조적 불가**(1-E): 우리가 요건을 못 갖춤(캐파·인증). → 관계 보존, "규모 되면 재접촉".
- **② 가치충돌/착취**(1-F): 하면 안 됨(카피·로열티 거부). → 관계 안 맺음, 정보 보호.

### 3.6 인바운드 전용 — 진위 게이트
- 통과/미통과 이분법이 모든 차원에 선행.
- **통과가 끝이 아니라 시작**(1-G) — 이후 실질 차원(정합·단계) 판정.
- 높은 Willingness가 함정일 수 있음 — 상대가 더 원해도 실질·규모 우선.

### 3.7 새 신호 축 후보 (데이터스키마 §8 — Open)
- **결정 구조**(1-D): 담당자 ≠ 결정권자. 협상 상대를 '내부 챔피언'으로 무장. 대기업·공공·CSR에서 반복. → 사업단계 하위신호로 흡수할지 별도 축으로 둘지 팀 결정.
- **서비스형 보완재**(1-J): 보완재가 상품이 아니라 역량·커리큘럼. resource_complementarity의 하위 유형화 검토.
